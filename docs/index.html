Project Overview

This project demonstrates a layered backend architecture: Controller - Service - Repository, with validation, authentication, and database persistence.

  • Spring Boot 2.7.14, Java 8
  • Spring Security + JWT for auth
  • Spring Data JPA + MySQL
  • RESTful APIs for User, Project, and Backlog Task

Architecture Flow

Client -> Security Filter -> Controller -> Validation -> Service -> Repository -> JPA/Hibernate -> MySQL -> Response

Protected APIs require header: Authorization: TMP <token>

Core API Endpoints

Authentication

  • POST /api/users/register
  • POST /api/users/login

Project

  • POST /api/project
  • GET /api/project/{projectId}
  • GET /api/project/all
  • PUT /api/project
  • DELETE /api/project/{projectId}

Backlog Task

  • POST /api/backlog/{projectId}
  • GET /api/backlog/{projectId}
  • GET /api/backlog/{projectId}/{projectSequence}
  • PUT /api/backlog/{projectId}/{projectSequence}
